Le brief

Petit jeu video realise en HTML, CSS et JavaScript dans le cadre d'un TP pour mes etudes, le but est simple, deux joueurs s'affrontent pour savoir qui des deux appuient le plus rapidement sur deux touches du clavier. L'idee ici est donc de faire le jeu dans l'univers de dragon ball, et de faire se combattre les deux personnages avec un kameha du celebre manga.

UN SYSTEME DE CHOIX DE PERSONNAGE

Un systeme de choix de personnage code avec des objects en JS qui nous permet de choisir le personnage voulu, cette fonction est purement esthetique, mais on peut imaginer son utilisation pour, par exemple changer la facon du joueur de jouer ou bien meme d'augmenter le nombre de click du joueur par click.

Systeme de combat

Systeme de combat entre les deux joueurs, celui qui appuie le plus voit son kameha detruire le joueur ennemi.
