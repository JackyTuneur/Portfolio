Le brief

Petit jeu video realise en HTML, CSS et JavaScript dans le cadre d'un TP pour mes etudes, le but etant relativement simple, faire apparaitre 64 cartes de dos et retrouver l'intru cache au hasard parmis elle, avec un systeme de difficulte allant de 64 cartes a 75 cartes. Je décide donc de partir sur une DA avec un gobelin que l'on doit trouvé.

UN SYSTEME DE TABLEAU DE CARTE

Un systeme de tableau de cartes qui s'affiche a l'interieur d'une balise "flex" en html, pour arriver a cela, je boucle sur le tableau de mes cartes pour ensuite l'afficher dans le DOM

Systeme de score

Tableau de joueur avec leurs score enegistre dans le Local Storage du navigateur.

CODE POUR RETOURNER LES CARTES

Cette fonction est l'une des plus importantes de mon code, elle permet de regarder si la carte est un intru ou pas en fonction de son "ID" (numéro d'identification)
